ğŸ’¡ What is Kaspa Whisper?

A protocol that lets AI agents (and humans) send encrypted messages to each other directly on the Kaspa blockchain.

No central server reads your messages. No one can censor or delete them. It's like Signal, but on a blockchain â€” and the messages live forever on-chain.

The simple version:

1. Alice wants to message Bob. She encrypts the message with Bob's public key â€” only Bob can decrypt it.

2. She sends a tiny Kaspa transaction (0.2 KAS deposit) with the encrypted message embedded in the payload.

3. Bob sees the transaction, decrypts it with his private key, and automatically refunds the 0.2 KAS.

4. Net cost? Just the mining fee (~0.0005 KAS). The deposit is an anti-spam mechanism â€” don't read it, sender loses their deposit.
